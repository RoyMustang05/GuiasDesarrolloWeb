Datos generales del curso

Curso: Desarrollo Web

Durante este curso exploramos a fondo cómo construir sitios web desde cero, combinando diseño, estructura y funcionalidad. Aprendimos los principios de HTML, CSS y JavaScript para crear páginas dinámicas y adaptables.

Temas principales: estructura de páginas web, estilos con CSS, interacción mediante JavaScript y diseño adaptable con las mediaqueries.

Guías realizadas

Guía 1: Fundamentos de HTML y CSS

En esta guía se trabajó la estructura básica de un documento HTML y la aplicación de estilos mediante CSS. Se practicó el uso de selectores, fuentes y propiedades visuales para dar formato a los textos y mejorar la presentación de los contenidos.

Guía 2: Selectores avanzados y modelo de caja

Esta guía profundizó en los selectores más complejos de CSS, incluyendo pseudo-clases y pseudo-elementos. Además, se aprendió a manejar el modelo de caja (margin, border, padding y content) para controlar mejor la distribución y el espaciado de los elementos.

Guía 3: Posicionamiento y maquetación web

En esta parte se experimentó con las distintas formas de posicionar elementos usando CSS (absoluto, relativo, fijo, etc.). También se aplicaron técnicas para diseñar páginas con estructuras flexibles, combinando secciones, menús y galerías.

Guía 4: Diseño responsivo y media queries

Esta guía se enfocó en el diseño adaptable para distintos dispositivos. Se aprendió a utilizar la metaetiqueta viewport, formularios con controles HTML5, y las media queries para que la página ajustara su apariencia según el tamaño de la pantalla.
